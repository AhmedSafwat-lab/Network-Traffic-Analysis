# Network Traffic Analysis using Wireshark

This project demonstrates a simple network traffic analysis using Wireshark.

## Objective

To capture and analyze network packets and identify potential security concerns.

## Tools Used

Wireshark
Kali Linux

## Traffic Generation

Network traffic was generated using the command:

ping 8.8.8.8

## Analysis

The captured packets were analyzed using Wireshark filters to observe ICMP traffic.
